## Overview

Skyplayr is a Chrome extension that provides Picture-in-Picture controls for supported HTML5 video players.

We do not collect, sell, or share personal data.

## Data We Collect

Skyplayr does not collect or transmit:

* Personally identifiable information
* Health information
* Financial and payment information
* Authentication information
* Personal communications
* Location data
* Web history
* User activity analytics
* Website content for storage or transfer

## Data Stored Locally

Skyplayr stores extension settings in Chrome extension storage so your preferences can be applied consistently.

Examples of settings:

* Show or hide the overlay button
* Auto-enter Picture-in-Picture on tab switch
* Restore Picture-in-Picture after autoplay

These settings are used only for extension functionality.

## Permissions and Why They Are Needed

Skyplayr requests the following permissions:

* storage: save and read user preferences
* activeTab: run a user-triggered Picture-in-Picture action on the active tab
* tabs: identify the active tab when handling toolbar and shortcut actions
* scripting: coordinate extension runtime behavior required for Picture-in-Picture controls
* host permissions (<all_urls>): detect and control HTML5 video elements on user-visited sites where Picture-in-Picture is requested

Skyplayr does not use permissions to collect or profile users.

## Data Sharing

Skyplayr does not sell user data.

Skyplayr does not transfer user data to third parties.

## Data Retention

Skyplayr keeps only extension preference values in Chrome storage until you change them, remove them, or uninstall the extension.

## Security

Skyplayr is designed to minimize data handling. No personal user data is transmitted to external servers by the extension.

## Children's Privacy

Skyplayr is not directed to children under 13 and does not knowingly collect personal data from children.

## Changes to This Policy

We may update this policy when extension behavior changes. Updates will be posted at this same URL.
